Notes for advanced users

  • Enchantments will default to the same quality as the item, at the highest enchanting level below the item level.
  • The Item IDs used in the Item Links were collected from UESP Mined Items collection and have not all been tested manually. Please check what is queued is correct before crafting valuable items.
  • When using Item Links, [Any Style] will default to Breton (Molybdenum) since it is the first style selected when at a crafting station.
  • Previewing the item links in-game will show the item as Transmuted to the desired trait. Dolgubon's Lazy Set Crafter will make the item with the intended trait as the base-trait correctly, but using the transmute trait field in the item links greatly reduces the number of item IDs needed for this page to work.
